# WeeChat - Online Chatting Application

WeeChat is a basic real-time online chatting application designed to connect users seamlessly. 

---

## Technologies Used

### Frontend
* React.js
* HTML5 & CSS3
* JavaScript (ES6+)
* DaisyUI
* TailwindCSS

### Backend
* Node.js
* Express.js
* Socket.IO

### Database
* MongoDB

---

## Steps to Run the Application

Follow these steps to get your local development environment up and running.

### 1. Prerequisites
Make sure you have **Node.js** and **npm** installed on your machine.

### 2. Clone the Repository
```bash
git clone [https://github.com/Sakirahmed61/react-chat-app.git](https://github.com/Sakirahmed61/react-chat-app.git)
cd react-chat-app
```

3. Start the Backend Server
From the root directory, install the dependencies and start the server:

```Bash
npm install
npm run server
```
4. Start the Frontend Application
Open a new terminal window, navigate to the frontend directory, install the dependencies, and start the development server:

```Bash
cd frontend
npm install
npm run dev
```
